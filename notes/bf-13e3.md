# NOAA Archive URLs for Lambert-Conformal (GDT 3.30) + DRT=3 GRIB2 Files

## Summary

Research completed for finding publicly accessible NOAA archive URLs hosting Lambert-conformal (GDT 3.30) GRIB2 files with complex packing (DRT=3/Data Representation Template 5.3).

## Key Technical Definitions

### GDT 3.30 (Grid Definition Template 3.30)
- **Template**: Lambert Conformal Conic projection
- **Usage**: Used by NOAA regional models (NAM, HRRR) over CONUS
- **Source**: [CPC wgrib2 documentation](https://ftp.cpc.ncep.noaa.gov/wd51we/wgrib2/grib2/intro_grids.doc) - "The Lambert conformal grid uses template 3.30"

### DRT 3 (Data Representation Type 3)
- **Template**: Data Representation Template 5.3
- **Method**: Grid point data with complex packing and spatial differencing
- **Source**: [WMO Manual on Codes Volume I.2](https://repository.oceanbestpractices.org/bitstreams/2dd4d5e9-f662-4944-854f-fda25eea4cdc/download)
- **Note**: "DRT 3" refers to Section 5, Template 3 (i.e., 5.3) in GRIB2 specification

## NOAA Models Identified with GDT 3.30 + Complex Packing

### 1. HRRR (High-Resolution Rapid Refresh) - 3km
- Resolution: 3km Lambert Conformal grid
- Grid dimensions: 1059 × 1799
- Central latitude: 38.5°N, Central longitude: 262.5°W
- **Primary recommended source** for current high-resolution data

### 2. NAM CONUS Nest - 12km/3km
- Resolution: 12km (Grid 218) and 3km high-resolution nest (Grid 227)
- Multiple CONUS domains available
- **NOTE**: NAM scheduled for retirement in early 2026, replaced by RRFS

## File Naming Convention

### HRRR Format:
```
hrrr.YYYYMMDD/conus/hrrr.tCCz.wrfsfcfXX.grib2
```
- `YYYYMMDD`: Date (e.g., 20250102)
- `CC`: Model cycle hour (00-23)
- `XX`: Forecast hour (F00-F48)

### NAM Format:
```
nam.YYYYMMDD/nam.tCCz.{product}XX.tm00.grib2
```
- `YYYYMMDD`: Date (e.g., 20241201)
- `CC`: Model cycle hour (00, 06, 12, 18)
- `XX`: Forecast hour (F00-F84)
- `{product}`: `awphys`, `awip12`, `conusnest.hiresf`, `goes218`

## Grid Definition and Data Representation Templates

### GDT 3.30 - Lambert Conformal Conic
**GDT 3.30** is the Grid Definition Template for **Lambert Conformal Conic** projection in GRIB2 files.

**HRRR Grid Specification:**
- Resolution: 3km
- Grid dimensions: 1059 × 1799
- Central latitude: 38.5°N
- Central longitude: 262.5°W (97.5°W)

**NAM Grid Specifications:**
- Grid 218 (AWIPS): 12km resolution CONUS
- Grid 227 (CONUS Nest): 3km high-resolution nest
- Projection: Lambert Conformal Conic for both

---

### DRT 3 / Template 5.3 - Complex Packing
**DRT 3** in GRIB2 refers to **Data Representation Template 5.3**:
- Grid point data with complex packing and spatial differencing
- Provides efficient compression for gridded meteorological data
- Commonly used by NOAA models (HRRR, NAM) for data compression

## Verification with wgrib2

These files can be verified for their grid definition and data representation using the `wgrib2` utility:

```bash
# Check Grid Definition Template (should show GDT=30 for Lambert Conformal)
wgrib2 -grid hrrr.t00z.wrfsfcf00.grib2

# Check Data Representation Type (should show DRT=3 for complex packing with spatial differencing)
wgrib2 -vt hrrr.t00z.wrfsfcf00.grib2

# Full inventory dump
wgrib2 -inventory hrrr.t00z.wrfsfcf00.grib2
```
