# NOAA GRIB2 Archive Hosting Platforms

**Bead ID:** bf-2p57  
**Date:** 2026-07-03

## Overview

NOAA provides public access to weather model GRIB2 data through multiple platforms and archives. This document catalogs the primary hosting platforms, their base URLs, and access requirements.

---

## 1. NOMADS (NOAA Operational Model Archive and Distribution System)

**Description:** The primary portal for accessing NOAA operational model data with near-real-time access.

**Base URLs:**
- Main server: `https://nomads.ncep.noaa.gov/`
- Data access: `https://nomads.ncep.noaa.gov/data/`

**Documentation:**
- Help page: https://nomads.ncep.noaa.gov/info.php?page=help
- Fast download guide: https://nomads.ncep.noaa.gov/info.php?page=fastdownload

**Models Available:**
- GFS (Global Forecast System)
- GDAS (Global Data Assimilation System)
- Various regional models

**Archive URL Pattern (GFS example):**
```
https://nomads.ncep.noaa.gov/data/nccf/com/gfs/prod/gfs.YYYYMMDDHH/
```

**Access Requirements:**
- No authentication required (public data)
- HTTP/HTTPS access
- Supports random access using index files for GRIB downloads

**Limitations:**
- Limited historical data retention (primarily operational data)
- For older historical data, use NCEI archives

---

## 2. NCEI (National Centers for Environmental Information)

**Description:** NOAA's official archive for long-term storage of weather and climate data.

**Base URLs:**
- Main portal: `https://www.ncei.noaa.gov/`
- Data access search: `https://www.ncei.noaa.gov/access/search/index`
- Archive services: `https://www.ncei.noaa.gov/archive`
- AIRS (Archive Information Request System): `https://www.ncei.noaa.gov/has/HAS.DsSelect`

**Models Available:**
- Historical weather model output
- Climate reanalysis datasets
- Specialized datasets

**Access Requirements:**
- Free account registration required for some services
- AIRS system requires login for order requests
- Public data search available without authentication

**Data Format:**
- Native archive formats (including GRIB2)
- Quality-controlled datasets

## 3. NOAA Big Data Program (NODD) - Cloud Platforms

### 3.1 AWS (Amazon Web Services)

**Registry Page:** https://registry.opendata.aws/collab/noaa/

**Available Models:**
- GFS (Global Forecast System)
- HRRR (High-Resolution Rapid Refresh)
- NAM (North American Mesoscale)
- RAP (Rapid Refresh)

**AWS Marketplace Listings:**
- GFS: https://aws.amazon.com/marketplace/pp/prodview-hok7o2o24ktfi
- GFS netCDF: https://aws.amazon.com/marketplace/pp/prodview-u4bxtndbzw4hk

**Historical Archive:**
- UCAR GDE: https://gdex.ucar.edu/datasets/d084001/
- GFS 0.25° global forecast grids (2021 to present)

**Quickstart Resources:**
- AWS GFS Quickstart Notebook: https://github.com/aws-samples/aws-opendata-samples/blob/main/notebooks/noaa-gfs/noaa_gfs_quickstart.ipynb

**Access Requirements:**
- No authentication for public buckets
- AWS account required (free to access data)
- Standard AWS egress fees may apply
- Trailing 30-day window for operational data

**Program Overview:** https://ncics.org/data/noaa-big-data-project/

### 3.2 Google Cloud Platform

**Program Announcement:** https://cloud.google.com/blog/products/data-analytics/weather-climate-big-data-from-noaa-now-in-cloud

**Available Models:**
- GFS, HRRR, RAP, NAM (same as AWS, through NOAA NODD program)

**Access Requirements:**
- Public cloud storage buckets
- GCP account required (free to access data)
- Standard egress/processing fees may apply

## 4. NOAA READY Gridded Data Archives

**Description:** NOAA ARL's archive system where NCEP model output is reprocessed and stored.

**Base URL:** https://www.ready.noaa.gov/archives.php

**Data Processing:**
- NCEP model output in GRIB format
- Reprocessed with 1-byte packing algorithm

**Access Requirements:**
- No authentication required
- HTTP/HTTPS access
- AWS CLI instructions provided for bulk downloads

---

## 5. NCEP Products Inventory

**Description:** Official inventory of NCEP data products.

**Base URL:** https://www.nco.ncep.noaa.gov/pmb/products/gfs/

**Products Available:**
- Global Forecast System (GFS) Model
- Global Data Assimilation System (GDAS) Model  
- GFS Wave Model

**Access Requirements:**
- No authentication required
- HTTP/HTTPS access

## Notes

1. **No Authentication Required for Most Public Data:** All platforms listed provide free public access without authentication requirements, with the exception of NCEI's AIRS system which requires account registration for ordering.

2. **Format:** All platforms serve data in GRIB2 format (some also offer netCDF conversions).

3. **Transition Note:** NOAA is transitioning from CDAS to CORe (Cloud Reanalysis) in March 2026. See: https://gribstream.com/blog/noaa-replaces-cdas-with-core-march-2026

4. **Archive vs. Operational:** NOMADS focuses on operational data (recent), while NCEI provides long-term archival storage.
