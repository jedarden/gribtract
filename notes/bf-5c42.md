## File Naming Convention

**Pattern:** `nam.tCCz.[category]fFF.tm00.grib2`

Where:
- `nam` = North American Mesoscale model
- `tCCz` = Model cycle runtime (CC = 00, 06, 12, 18)
- `category` = File category (conusnest, firewxnest, etc.)
- `fFF` = Forecast hour (00-60 for nests, 00-84 for CONUS)
- `tm00` = Time indicator (always tm00)
- `grib2` = GRIB2 format

**Index Files:** Each .grib2 file has a corresponding .idx file for subsetting:
- `nam.t00z.conusnest.hiresf00.tm00.grib2.idx`
