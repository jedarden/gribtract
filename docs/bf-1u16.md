# NOAA Models Using Lambert Conformal GDT 3.30

## Summary

This document summarizes NOAA weather models that use Lambert Conformal projection with Grid Definition Template (GDT) 3.30 in GRIB2 format.

## Grid Definition Template 3.30

**GDT 3.30** is the GRIB2 Grid Definition Template for **Lambert Conformal Conic** projections. According to [NOAA NCEP documentation](https://www.nco.ncep.noaa.gov/pmb/docs/grib2/grib2_doc/grib2_temp3-30.shtml):

- Template 3.30 encodes Lambert Conformal projections (can be secant, tangent, conical, or bipolar)
- Grid lengths are in units of 10^-3 m at the latitude specified by LaD
- If Latin 1 = Latin 2, the projection is on a tangent cone
- Created: June 29, 2005

Sources:
- [NOAA NCEP GRIB2 Template 3.30 Documentation](https://www.nco.ncep.noaa.gov/pmb/docs/grib2/grib2_doc/grib2_temp3-30.shtml)
- [CPC FTP intro_grids.doc](https://ftp.cpc.ncep.noaa.gov/wd51we/wgrib2/grib2/intro_grids.doc) - "The Lambert conformal grid uses template 3.30"
- [NOAA VLab DeGRIB Gotchas](https://vlab.noaa.gov/web/mdl/degrib-gotchas)

## NOAA Models Using Lambert Conformal GDT 3.30

### 1. HRRR (High-Resolution Rapid Refresh)

**Coverage:** CONUS (Contiguous United States)

**Resolution:** 3 km

**Projection Parameters** (from [NASA Technical Report](https://ntrs.nasa.gov/api/citations/20160009371/downloads/20160009371.pdf)):
- Central Latitude: 38.5°N
- Central Longitude: 262.5°W (97.5°W)
- Projection: Lambert Conformal

**Forecast Hours:** 0-18 hours
- Hourly updates
- Sub-hourly surface fields (15-minute intervals) out to 18 hours

**Purpose:** Real-time cloud-resolving, convection-allowing atmospheric model for short-term weather forecasting

Sources:
- [NOAA HRRR Official Page](https://rapidrefresh.noaa.gov/hrrr/)
- [AWS Open Data Registry - HRRR](https://registry.opendata.aws/noaa-hrrr-pds/)
- [GribStream Blog - HRRR sub-hourly fields](https://gribstream.com/blog/hrrr-sub-hourly-15-minute-surface-fields-now-available)

### 2. NAM CONUS (North American Mesoscale Model)

**Coverage:** CONUS (Contiguous United States)

**Resolution:** Multiple grids available:
- 3 km CONUS nest (1792 x 1059)
- 12 km Lambert CONUS grid
- 29 km NAM Model (Quadruple Resolution)

**Projection:** Lambert Conformal Conic

**Forecast Hours:** 0-48 hours for 3 km CONUS nest

**Purpose:** Regional weather prediction over North America with multiple nested domains

Sources:
- [NOAA NAM on AWS](https://registry.opendata.aws/noaa-nam/)
- [READY Gridded Forecast Data](https://www.ready.noaa.gov/READYmetdata.php) - NAM 3 km CONUS nest documentation
- [GribStream - NAM datasets](https://gribstream.com/blog/noaa-nam-datasets-now-available)
- [LuckGrib - NAM CONUS](https://luckgrib.com/models/nam_conus/)

### 3. RAP (Rapid Refresh)

**Coverage:** CONUS (Contiguous United States)

**Resolution:** 13 km (also available at 40 km regional CONUS pressure levels)

**Projection:** Lambert Conformal Conic

**Forecast Hours:** Updated hourly with short-range forecasts

**Purpose:** High-frequency weather prediction system providing hourly analysis and forecasts

Sources:
- [NOAA NCEI RAP Metadata](https://www.ncei.noaa.gov/access/metadata/landing-page/bin/iso?id=gov.noaa.ncdc:C00690)
- [AWS Open Data Registry - RAP](https://registry.opendata.aws/noaa-rap/)
- [NCEP RAP Products](https://www.nco.ncep.noaa.gov/pmb/products/rap/)
- [Herbie Documentation](https://herbie.readthedocs.io/en/latest/gallery/noaa_models/rap.html)

### 4. RRFS (Rapid Refresh Forecast System)

**Coverage:** CONUS (Contiguous United States)

**Resolution:** 3 km (1799 x 1059 grid)

**Projection:** Lambert Conformal CONUS

**Forecast Hours:**
- Hourly updates: 0-18 hours
- Extended forecasts (00Z, 06Z, 12Z, 18Z cycles): 0-84 hours

**Purpose:** Next-generation rapidly updating high-resolution weather model (scheduled operational ~August 2026)

Sources:
- [NOAA GSL RRFS Page](https://gsl.noaa.gov/rrfs/)
- [NCEP RRFS Products](https://www.nco.ncep.noaa.gov/pmb/products/rrfs/)
- [GribStream - RRFS replaces NAM](https://gribstream.com/blog/rrfs-replaces-nam-hiresw-href-narre-proposal)

### 5. NDFD (National Digital Forecast Database)

**Coverage:** CONUS, Puerto Rico, Hawaii, Guam

**Resolution:** Multiple grids:
- 2.5 km NDFD grid (CONUS)
- 5 km NDFD grid (CONUS)
- 3 km NDFD grid (various regions)

**Projection:** Lambert Conformal (using GDT 3.30)

**Purpose:** Gridded weather forecasts combining automated guidance and forecaster input

Sources:

## Conclusion

The following NOAA regional models are confirmed to use Lambert Conformal projection with GRIB2 Grid Definition Template 3.30:

1. **HRRR** - 3 km CONUS, 0-18 hour forecasts
2. **NAM CONUS** - Multiple resolutions (3/12/29 km), 0-48 hour forecasts
3. **RAP** - 13 km CONUS, hourly updates
4. **RRFS** - 3 km CONUS, 0-18 hourly / 0-84 extended forecasts
5. **NDFD** - 2.5/3/5 km grids, gridded forecasts

All of these models serve the CONUS domain with Lambert Conformal projections optimized for mid-latitude weather forecasting over the United States.
